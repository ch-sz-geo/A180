== Modelltransformationen
=== Transformation von der Struktur der Datenerfassung ins kantonale Datenmodell
* Quellmodell: Als Input für das "Quellmodell" dient eine Excel-Liste, in welcher der Auftragnehmer Attributerweiterungen vorschlägt.
* Zielmodell: Das hier beschriebene Datenmodell

==== Klasse `+Bauwerksgruppe+`
Die Klasse `+Bauwerksgruppe+` dient der Zusammenfassung von Bauwerken zu einer Hindernisgruppe. Im Quellsystem sind die Bauwerkskennungen als Komma-separierte Liste geführt. Bei der Überführung ins kantonale Modell wird eine Bauwerksgruppe erstellt. Die Gruppenzuordnung erfolgt über das Attribut `+rGruppe+` der Klasse `+Bauwerksbewertung+`.
